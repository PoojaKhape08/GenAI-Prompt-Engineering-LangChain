# GenAI Prompt Engineering using LangChain

## Overview
This project focuses on designing reusable and dynamic prompt systems using LangChain instead of hardcoded prompts.

##  Features
- PromptTemplate implementation
- Multi-input prompt handling
- Prompt variations (Teaching, Interview, Storytelling)
- ChatPromptTemplate system
- Input validation layer
- Mini Prompt Generator App
- Template reusability testing

##  Tech Stack
- Python
- LangChain
- Jupyter Notebook

##  Outcome
Built a complete prompt pipeline:
User Input → Validation → Template → Output
